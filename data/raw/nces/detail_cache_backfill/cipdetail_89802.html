Detail for CIP Code 26.1103

  • Print

Title: Bioinformatics.

Definition: A program that focuses on the application of computer-based technologies and services to biological, biomedical, and biotechnology research. Includes instruction in algorithms, network architecture, principles of software design, human interface design, usability studies, search strategies, database management and data mining, digital image processing, computer graphics and animation, CAD, computer programming, and applications to experimental design and analysis and to specific quantitative, modeling, and analytical studies in the various biological specializations. See also: 11.0104) Informatics., 51.2706) Medical Informatics.

Action: No Substantive Changes
