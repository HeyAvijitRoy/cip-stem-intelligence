# CIP STEM Overlay

An open-source tool that overlays **DHS STEM OPT eligibility** onto official **NCES CIP definitions**, providing searchable, filterable, and versioned insight into STEM-designated degree programs.

This project does **not** replace official DHS or NCES resources. It adds an intelligence layer to make STEM eligibility easier to discover, audit, and understand.

---

## What This Project Does

- Combines NCES CIP definitions with DHS STEM eligibility lists
- Provides a static, searchable interface (GitHub Pages)
- Tracks DHS STEM list changes over time
- Publishes verifiable, versioned datasets
- Automates updates using GitHub Actions

---

## Authoritative Sources

- NCES CIP Taxonomy & Definitions
- DHS / ICE STEM Designated Degree Program List (PDF)

The DHS STEM list remains the **only authoritative source** for STEM OPT eligibility.
